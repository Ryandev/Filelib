#Filelib
**cross platform c-api for file access**

clear api's to read/write files to and from the disk.

##Usage

```
$cd $filelib_dir/example;
chmod +x example_run.sh;
./example_run.sh
```

##Validate

```
$cd $filelib_dir/test;
chmod +x test_run.sh;
./test_run.sh
```
